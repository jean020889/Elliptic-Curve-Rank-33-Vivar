Overview
This repository documents the identification of an elliptic curve over the rational field \mathbb{Q} with an analytic rank of 33. This discovery establishes a new historical benchmark, surpassing previous records and providing new data for the study of the Birch and Swinnerton-Dyer (BSD) conjecture.
The research was conducted by Vivar (Jean Carlos Vivar Benítez) as part of an independent investigation into high-genus arithmetic surfaces.
